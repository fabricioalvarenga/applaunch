# AppLaunch

A macOS simple application launcher inspired by the native macOS Launchpad.

## About the Project

This project is a macOS application launcher created specifically for modern
versions of macOS where the native Launchpad has been removed.

## Current Features

- Display installed applications in a grid layout
- Fast app launching
- Native macOS user interface

## Roadmap / Planned Features

The features listed below are not yet implemented. They may be developed in future versions or through
community contributions.

- [ ] Open and close animations
- [ ] Window overlaying the macOS menu bar
- [ ] Support for multiple displays
- [ ] Ability to create folders to group applications
- [ ] Drag-and-drop reordering of applications and folders
- [ ] Scroll the application grid using mouse drag gestures

## Requirements

- macOS XX or later
- Xcode XX or later

## How to Run

1. Clone this repository
2. Open the `.xcodeproj` file in Xcode
3. Build and run the application
